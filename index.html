How to Train

Mode 1 — Triadic Entailment

Each trial contains exactly two spatial premises and one tested conclusion using three letter-bound objects. Example: “A is west of J; J is north of P; P is southeast of A.”

Construct the relational organisation created by the first two statements. Press MATCH only when the third statement is logically entailed by that organisation. In the example, A is northwest of P, so P is southeast of A and the third statement is a MATCH.

The letters are computational nodes, not decoration. Reversing a letter binding, changing which letter is the shared middle node, or confusing subject and object changes the logical result. The engine recomputes the answer from the letter graph after generation rather than trusting the intended answer branch.

Directional meta-distinction

Mode 1 uses a sixteen-way directional ontology: north, north-northeast, northeast, east-northeast, east, east-southeast, southeast, south-southeast, south, south-southwest, southwest, west-southwest, west, west-northwest, northwest and north-northwest.

Higher interference makes false conclusions approach the exact answer. A conclusion one 22.5° step away remains NO MATCH even when it is broadly compatible with the same coarse direction. Premise order and wording may be inverted without changing the underlying relation.

Mode 1 is a within-trial entailment task, so its inference structure is fixed at two premises plus one tested conclusion. The N-back slider is therefore disabled in Mode 1.

Mode 2 — Ontological Category Integration

The previous Mode 1 system is preserved here. Each sentence contains three independently transformed symbols. Their ontology categories and forms create three node transformations; two cardinal transitions bind the roles; and the transition between those transitions creates a higher-order turn.

The nine categories are All, Difference, Action, Division, Connection, Multiplication, Projection, Encompassment and Completion. N-back scoring compares the derived integrated ontology and integrated form, while complete triadic logic drives cognitive interference. Letters remain surface carriers in Mode 2.

Cognitive interference

In Mode 1, interference controls directional resolution, premise inversion, clause order and near-miss conclusions. In Mode 2, it controls near-miss category bindings, topology, synthesis outcomes and proactive interference from intervening triads.

Accessibility guarantee

The program constructs one final premise string, writes that exact string to the live visual region and sends the same string to speech synthesis. The response window opens only after speech finishes; audio-only display, pause/resume replay, speech rate, volume and speech/background separation remain available.

This is a theoretically motivated relational-integration design. It is not validated evidence that training will increase general fluid intelligence.
