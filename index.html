How to Train

Mode 1 — Triadic Entailment

Every trial contains exactly three relational statements using three letters. The first two statements construct the relation among the letters. The third statement is part of the triad’s logical profile: it may be exactly entailed, approximately related, reversed, attached to the wrong letters or valid under a different rule.

Choose N-back 1 through N-back 8. After the initial memory-fill trials, press MATCH when the current triad preserves the logical components required by the active comparison criterion relative to the triad N positions back. Press NO MATCH when a criterion-relevant component changes.

Letter identity never defines a match. Consistent renaming may change every letter while preserving the match. Absolute direction may also change through a lawful rotation, reflection or inverse reformulation. Repeated lettering and similar wording do not create a match when binding, proof, modality, precision, reference frame, criterion or rule regime changes.

The visible triad contains only its three relations. It never displays a contract heading, rule label, answer explanation or “therefore.” Match and non-match explanations remain separate and appear only after the player responds.

N-back levels 1–8

Level 1 compares with the immediately preceding triad. Level 8 compares with the triad eight positions back. The first N triads fill the memory buffer and are not scored. Each level contains twenty distinct approved comparison archetypes: ten MATCH and ten NO MATCH, producing 160 operational cases in total.

Level 1 regulates direct entailment, binding and local-versus-global validity. Level 2 regulates inversion, renaming and endpoint roles. Level 3 regulates four-, eight- and sixteen-way precision. Level 4 regulates metric assumptions and model-set modality. Level 5 regulates rotation, reflection, chirality and reference frames. Level 6 separates same answers from identical or equivalent proofs. Level 7 regulates evaluation policies and criteria. Level 8 integrates proof, contract, counterfactual dependency and minimal rule revision.

Meta-distinction and meta-logic

The comparison profile can include binding structure, proof topology, proof identity, proof equivalence, composition mechanism, equal-unit versus unspecified-distance reasoning, directional resolution, modal force, precision, query orientation, reference frame, chirality, evaluation policy, criterion, rule regime, revision state and counterfactual dependency.

Matches preserve the components required by the active comparison criterion. Non-matches are generated as adjacent logical alternatives: exact versus coarse, necessary versus merely possible, correct endpoint versus locally true but globally irrelevant, same answer versus different proof, rotation versus reflection, stable rule versus minimally revised rule, or one criterion of validity versus another.

Mode 2 — Ontological Integration

The previous ontology-category system is preserved here. Each sentence contains three independently transformed symbols. Their ontology categories and forms create three node transformations; two cardinal transitions bind the roles; and the transition between those transitions creates a higher-order turn.

The nine categories are All, Difference, Action, Division, Connection, Multiplication, Projection, Encompassment and Completion. N-back levels 1–8 compare the derived integrated ontology and integrated form, while complete triadic logic drives cognitive interference. Letters remain surface carriers in Mode 2.

Accessibility guarantee

The program constructs one final three-statement string, writes that exact string to the live visual region and sends the same string to speech synthesis. The response window opens only after speech finishes; audio-only display, pause/resume replay, speech rate, volume and speech/background separation remain available.

This is a theoretically motivated relational-integration design. It is not validated evidence that training will increase general fluid intelligence.
